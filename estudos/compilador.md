## Instalando o Compilador Oficial

Dentro de um projeto npm execute:

~~~bash
npm install typescript --save-dev
~~~

## Executando o Compilador

~~~bash
npx tsc
~~~

## Configurando o Compilador

- Criando o arquivo tsconfig.json:

~~~bash
npx tsc --init
~~~
